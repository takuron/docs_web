开始使用 Docker

什么是 Docker?

Docker 是一个开放平台，用于开发、交付和运行应用程序。它允许您将应用程序与基础架构分离，以便可以快速交付软件。通过 Docker，您可以像管理应用程序一样管理基础架构，通过利用 Docker 的方法来快速交付、测试和部署代码，可以显著减少编写代码和在生产环境中运行代码之间的延迟。

从社区源安装 (Linux)

使用官方提供的便捷脚本可以快速安装 Docker Engine：

# 1. 下载并运行安装脚本
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh

# 2. (推荐) 将当前用户添加到 docker 组以免 sudo
sudo usermod -aG docker $USER

# 3. 重新登录或重启终端以使组设置生效

交互式命令速查

点击分类筛选，或展开卡片查看更多选项与说明。

拉取镜像

从 Docker Hub 或其他镜像仓库下载镜像。

docker pull <镜像名>:<标签>
示例与选项

例如，拉取最新版的 Ubuntu：
docker pull ubuntu:latest

不指定标签则默认为 `latest`。

列出本地镜像

显示所有已经下载到本地的镜像。

docker images

删除本地镜像

从本地删除一个或多个指定的镜像。

docker rmi <镜像ID或镜像名>
示例与选项

强制删除（即使有容器正在使用）：
docker rmi -f <镜像ID>

构建镜像

根据 Dockerfile 构建一个新的镜像。

docker build -t <镜像名>:<标签> .
示例与选项

-t 用于给镜像命名和打标签。

. 表示 Dockerfile 在当前目录。

创建并运行容器

使用指定镜像创建一个新容器并启动它。

docker run [选项] <镜像名> [命令]
常用选项
  • -d: 后台运行
  • -p 宿主端口:容器端口: 端口映射
  • -v 宿主路径:容器路径: 数据卷挂载
  • --name <名字>: 指定容器名
  • --rm: 容器停止后自动删除
  • -it: 交互式运行 (进入 shell)

列出容器

显示正在运行的容器。

docker ps
示例与选项

显示所有容器（包括已停止的）：
docker ps -a

停止/启动/重启容器

管理正在运行的容器。

docker stop <容器ID或名字>
其他命令

启动一个已停止的容器：
docker start <容器ID或名字>

重启容器：
docker restart <容器ID或名字>

删除容器

删除一个或多个已停止的容器。

docker rm <容器ID或名字>
示例与选项

强制删除一个正在运行的容器：
docker rm -f <容器ID或名字>

查看容器日志

获取容器的标准输出日志。

docker logs <容器ID或名字>
示例与选项

持续跟踪日志输出：
docker logs -f <容器ID或名字>

进入容器执行命令

在一个正在运行的容器内部执行命令。

docker exec -it <容器ID或名字> /bin/bash

启动 Compose 应用

根据 `docker-compose.yml` 文件在后台创建并启动所有服务。

docker compose up -d
示例与选项

在前台启动并显示日志：
docker compose up

如果需要，在启动前重新构建镜像：
docker compose up -d --build

停止 Compose 应用

停止并删除由 `up` 创建的容器、网络。

docker compose down
示例与选项

在停止时一并删除数据卷：
docker compose down -v

查看 Compose 服务状态

列出 Compose 应用中的所有容器。

docker compose ps

查看 Compose 日志

查看所有服务的日志。

docker compose logs
示例与选项

持续跟踪日志输出：
docker compose logs -f

查看指定服务的日志：
docker compose logs -f <服务名>

在服务中执行命令

在指定的运行中服务容器内执行命令。

docker compose exec <服务名> <命令>

清理系统

删除所有未使用的容器、网络、镜像（悬空镜像）等。

docker system prune
示例与选项

警告：这是一个清理命令，会删除数据。

删除所有未使用的本地镜像（不仅是悬空镜像）：
docker system prune -a
